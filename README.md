# Sorting Visualizer

## Demo
https://patrickdicks.github.io/SortingVisualizer/

## About
Sorting Visualizer allows for the visualization of various sorting algorithms.

* Supported Sorting Algorithms:
  * Merge Sort
  * Quick Sort
  * Heap Sort
  * Bubble Sort
  * Insertion Sort
  * Selection Sort 

* Technologies Used:
   * CSS
   * HTML
   * JavaScript
   * React
   * Semantic UI

## Installation:
```
npm install
```

## Run:
```
npm start
```
